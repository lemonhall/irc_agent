# 背景音乐目录

将你的背景音乐文件放在这里。

## 支持的格式

- `.mp3` - MP3 格式
- `.wav` - WAV 格式
- `.m4a` - M4A 格式
- `.flac` - FLAC 格式
- `.ogg` - OGG 格式

## 推荐音乐风格

适合新闻播报的背景音乐：

- 🎹 轻柔的钢琴曲
- 🎻 轻音乐、环境音乐
- 🎼 新闻背景音乐专用素材
- 🎧 Lofi、Chillout 风格

## 注意事项

- ⚠️ 音乐会自动循环播放，匹配人声长度
- ⚠️ 默认音量为 15%（可通过 --volume 参数调整）
- ⚠️ 建议使用无版权或已获授权的音乐
- ⚠️ 避免选择节奏过于强烈或有明显人声的音乐
